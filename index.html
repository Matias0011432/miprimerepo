Esto es una página básica para mostrar los fundamentos de HTML y CSS. El sitio web incluye una estructura simple con un encabezado, contenido principal, una sección de información personal, como punto de partida para realizar personalizaciones y añadir nuevas funcionalidades.

El objetivo de este proyecto es proporcionar una base para entender la estructura básica y familiarizarse con la organización del contenido en bloques y el uso de hojas de estilo para definir la apariencia visual de cada elemento. Al trabajar sobre este ejemplo concreto, van a poder facilitar la comprensión de conceptos como la jerarquía de elementos y la separación entre estructura y diseño.
